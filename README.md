# 💻 Code Editor Web App

A full-stack web-based code editor that allows users to write, edit, and manage code efficiently in a browser environment. This project combines a modern React frontend with a backend server for handling logic and requests.

---

## 🚀 Features

* ✍️ Real-time code editing
* 🎨 Clean and responsive UI
* 🔐 Backend integration for handling requests
* ⚡ Fast and lightweight performance
* 📂 Organized project structure (frontend + backend)

---

## 🛠️ Tech Stack

### Frontend

* React.js
* HTML5
* CSS3
* JavaScript

### Backend

* Node.js
* Express.js

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/Pateltushar9951/code-editor.git
cd code-editor
```

---

### 2️⃣ Install Frontend Dependencies

```bash
npm install
```

---

### 3️⃣ Install Backend Dependencies

```bash
cd server
npm install
```

---

### 4️⃣ Run the Application

#### ▶️ Start Backend

```bash
cd server
npm start
```

#### ▶️ Start Frontend

```bash
cd ..
npm start
```
